Data Migration 1.1.0 to 1.5.0
=============================
1) Please execute rss_manager_migration.sql script against to WSO2_RSS_DB database (Please find the latest script at https://docs.wso2.com/display/SS150/Upgrading+from+a+Previous+Release )
2) Please execute cassandra_permissions_migration.sql script against to WSO2_CARBON_DB database to migrate from Storage Server 1.1.0 to 1.5.0
